# Color-Palette

**Color-Palette** is a simple web application built using **HTML, CSS, and JavaScript**.

## Project Description
This app allows you to display different colors and generate color palettes easily.  
It can be used for design purposes or choosing colors for your projects.

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Use
1. Open the `index.html` file in your web browser.
2. Interact with the app to explore different colors and palettes.

## Note
This project is a frontend-only application and does not require any server to run.
